# Help Videos Directory

Place your instructional video files here with the following names:

- addition.mp4 - Video explaining addition
- subtraction.mp4 - Video explaining subtraction
- multiplication.mp4 - Video explaining multiplication
- division.mp4 - Video explaining division
- times-tables.mp4 - Video explaining times tables

If a video file is not present, the app will display friendly fallback text instead.

Video format: MP4 (H.264 codec recommended)
Recommended resolution: 720p or 1080p
Keep file sizes reasonable for web delivery (under 50MB per video)
